| 属性                         | 说明                                   |
| ---------------------------- | -------------------------------------- |
| column-width: 200px;         | 栏目宽度                               |
| column-rule: 5px dashed red; | 栏目边框                               |
| column-gap: 28px;            | 栏目间距                               |
| column-count: 7;             | 栏目数，不能与column-width属性同时使用 |

多用于对于文字的排列，需要将文字分成好几列内容的时候使用。
