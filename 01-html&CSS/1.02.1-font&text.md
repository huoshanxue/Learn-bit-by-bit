#### 1. 字体font

```css
font-family: "consolas","Arial","Microsoft Yahei","SimSun";

font-size: 1px;

color: #252525;

line-height: 28px;

font-weight: bold;

font-style: normal;  正常

font-style: italic; /*斜体。如果字体存在倾斜效果就倾斜，不存在就换成有倾斜效果的字体 */

font-style: oblique; /*斜体。给字体倾斜 */

font: bold italic 26px/50px "Arial"; /* font合并书写,
分别为：font-weight\style\size\line-height\font-family */

```

#### 2. 文本text

``` css
text-align: left;  /* 居左 */

text-align: center;  /* 居中 */

text-align: right;    /* 居右 */

text-decoration: none;  /* 无下划线 */

text-decoration: underline;  /* 下划线 */

text-decoration: line-through;   /* 删除线 */

text-decoration: overline;  /* 上划线 */

text-indent: 2em; /* \80px\10%  缩进，多用第一个，两个中文字 */ 
```
