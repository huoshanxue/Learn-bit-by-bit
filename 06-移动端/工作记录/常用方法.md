## 判断安卓还是 ios

> 注释：`match()`方法是用来检索字符串内指定的值，`str.match(regexp)`，如果字符串能够匹配到正则表达式就返回一个数组，如果没有就返回 null。

```js
if (navigator.userAgent.match(/(iPhone|iPod|iPad);?/i)) {
  // ios
} else if (navigator.userAgent.match(/andriod/i)) {
  // android
}
```
